# HealTrack
HealTrack est une application web dédiée à la réhabilitation à distance des patients. Elle propose des programmes d'exercices personnalisés et un suivi en temps réel des progrès, le tout dans un environnement sécurisé grâce à Symfony. Les professionnels de santé peuvent ainsi suivre facilement l'évolution de leurs patients. Projet WorkShop EPSI I1
